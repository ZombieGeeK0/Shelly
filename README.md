# Shelly

![Shell](https://github.com/ZombieGeeK0/Shelly/assets/158185295/ba0673ee-f485-4443-a459-c65082f36f91)

`Shelly es una Reverse Shell para la ejecución de comandos de forma remota hecha con Python, y su código ha sido ofuscado, de forma que es difícil de detectar por los antivirus convencionales. Proporciona ejecución remota de comandos en cualquier OS. Es muy intuitiva y manejable, además de ligera e indetectable.`
